# Thurston County Environmental Data Analysis

Geospatial and statistical analysis of **Thurston County, WA wetlands and parks data** using R. This project visualizes environmental land-use patterns, park coverage, and wetland distribution across the county.

## Overview

This project explores publicly available GIS and environmental datasets for Thurston County, Washington — applying data wrangling, geospatial mapping, and statistical analysis techniques to understand land conservation and green space distribution.

## Analysis Highlights

- Wetland classification and distribution mapping
- Park coverage by area and location
- Spatial correlation between parks and wetland zones
- County-wide environmental land-use summary
